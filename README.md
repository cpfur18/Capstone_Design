# 💻 노트북 추천 웹 서비스

<br>

## 📌 프로젝트 소개
- 사용자의 사용목적 및 성향을 수집 한 후, 추천 알고리즘을 사용하여 노트북을 구매하고자 하는 사람에게 적합한 노트북을 추천할 수 있는 반응형 웹 사이트를 개발한다.

<br>

## ❗ 개발 필요성
- 코로나 19로 인한 노트북 수요 증가.
- 방대한 정보로 인해 노트북 구매에 어려움을 겪는 소비자.
- 인기 상품 뿐 아니라 소비자 취향에 맞는 다양한 상품 노출 가능.

<br>

## ❕ 기술적 차별성
- 제품 추천을 위한 질의응답 : 사용자의 사용용도, 취향, 작업수준에 맞는 노트북을 질의응답을 통해 정보를 수집, 분석하여 맞춤형 정보를 제공.
- 컴퓨터 지식이 부족한 사용자를 고려한 UI : 컴퓨터 용어를 잘 모르는 사용자들도 키워드를 누르면 이 용어가 어떤 의미인지 직관적 알 수 있도록 표현.
- 알기 쉬운 성능 표기 : 노트북의 스펙을 수치화 한 뒤, 한 눈에 알아보기 쉽도록 가공하여 누구나 이해할 수 있도록 표시 할 수 있음.
